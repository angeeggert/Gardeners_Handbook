## 1. What is it?
The Valuation Prism is a logic protocol that teaches an AI to judge the **value** of a text block based on meaning, not just recency.
Instead of deleting old data randomly (FIFO), it scans content through specific "Lenses" to determine if it is **Signal (Keep)** or **Noise (Prune)**.

**It introduces the "Inversion of Proof":** The default state of data is "Keep". Only if it fails all resonance checks is it marked for deletion.
